# Whitepace

A frontend landing page designed to learn and demonstrate responsive web design principles.

## Technologies Used

- HTML5
- CSS3 (Responsive Design)
- Font Awesome icons
- Google Fonts

## Project Structure

```
Whitepace/
├── index.html              # Main landing page
├── README.md               # Project documentation
└── assets/
    ├── css/
    │   ├── style.css       # Main stylesheet
    │   └── responsive.css  # Responsive design styles
    └── img/                # Images and icons
```

## Getting Started

Simply open `index.html` in your web browser to view the responsive landing page.
